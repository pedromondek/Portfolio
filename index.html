    Do Excel à Análise de Dados
    Criação e manipulação de um Dataframe através de uma planilha (.docx) com dados reais financeiros da Bolsa de Valores (2023). Análise de Dados feita em Python utilizando frameworks para geração gráfica e análitica, utilizando a API yFinance e prophet.
    Do Excel à Análise de Dados

    O projeto se inicia com uma planilha com uma tabela de ações da Bolsa de Valores brasileira (2023-2024), possuindo suas variações de valores no tempo, volume, valor máximo e mínimo. Complementamos com mais informações (idade da empresa e segmento) com auxílio do ChatGPT (4.0). Realizamos uma série de análises das informações, que você pode conferir completo no repositório do projeto, segmentando as variações e comparando-os, suas variações de acordo com idade das empresas, etc.


    Utilizando Python, importando a planilha e bibliotecas Python (operator, turtle, pyparsing, pandas, plotly.express), transformamos adequadamente o arquivo (.docx) para a manipulação, mas apenas aprendizado, sem mais sua utilização. Adicionamos mais as bibliotecas matplotlib, mplfinance, yfinance, e utilizaremos os data frames da yfinance, estruturamos os dados e criamos os gráficos para análise.


    O primeiro gráfico, este linear, ilustra o desempenho anual de uma empresa através de seu fechamento diário (2023). O segundo trata-se de um gráfico de velas (referente a mesma empresa), com valor máximo e mínimo diário (2023), uma média móvel de sete dias e outra de quatorze dias. No terceiro gráfico, melhoramos o segundo, adicionando na sua parte inferior um gráfico de barras representando o volume transacionado, e tornando-o interativo.


    Por último, adicionamos mais uma biblioteca, o prophet, que utilizaremos para realizar algumas previsões de dados futuros de acordo com os dados atuais (2020-2023) e fazendo uma comparação com nossos dados de treinamento.
